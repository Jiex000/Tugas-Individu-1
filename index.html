HTML

HTML (Hyper Text Markup Language) adalah bahasa standar yang digunakan untuk membuat halaman web dan aplikasi web. HTML digunakan untuk menyusun dan mengatur konten di dalam halaman web, seperti teks, gambar, tautan, tabel, formulir, dan elemen-elemen lainnya.

Struktur dasar HTML:

  1. <!DOCTYPE html> : Memberitahu browser bahwa dokumen ini menggunakan HTML.
  2. <html> : Elemen root dari dokumen HTML.
  3. <head> : Berisi informasi meta tentang dokumen.
  4. <title> : Judul halaman yang ditampilkan di tab browser.
  5. <body> : Berisi konten utama halaman web.

Tags dasar HTML:

  • <h1> sampai <h6> : Heading / judul.
  • <p> : Paragraf.
  • <a> : Hyperlink.
  • <img> : Gambar.
  • <ul>, <ol>, <li> : Daftar.
  • <div> : Membungkus elemen untuk styling.

HTML

HTML (Hyper Text Markup Language) adalah bahasa standard yang digunakan untuk membuat halaman web dan aplikasi web. HTML digunakan untuk menyusun dan mengatur konten di dalam halaman web, seperti teks, gambar, tautan, tabel, formulir, dan elemen-elemen lainnya.

Berikut adalah struktur dasar HTML :

  1. < !DOCTYPE html > : Ini memberitahu browser bahwa dokumen ini menggunakan HTML.

  2. < html > : Elemen root dari dokumen HTML.

  3. < head >: Bagian ini berisi informasi meta tentang dokumen, seperti judul halaman.

  4. < title >: Judul halaman yang ditampilkan di tab browser.

  5. < body >: Bagian ini berisi konten utama halaman web, seperti teks, gambar, dan elemen lainnya.


Dalam coding HTML, akan menggunakan berbagai tags teks seperti :

  • < h1 > sampai < h6 >: Digunakan untuk membuat heading atau judul dengan tingkat kepentingan yang berbeda.

  • < p >: Digunakan untuk membuat paragraf teks.

  • < a >: Digunakan untuk membuat tautan (link) ke halaman lain atau sumber daya lainnya.

  • < img >: Digunakan untuk menyisipkan gambar ke dalam halaman web.

  • < ul >, < ol >, dan < li >: Digunakan untuk membuat daftar tak berurutan (unordered list) dan daftar berurutan (ordered list).

  • < div >: Digunakan untuk mengelompokkan elemen-elemen HTML bersama-sama, sering digunakan untuk tujuan styling dengan CSS.
